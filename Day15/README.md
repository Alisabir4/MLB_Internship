## Project Overview

This project implements an image classification system using a **Convolutional Neural Network (CNN)** with TensorFlow/Keras. The model is trained on the Fashion MNIST dataset to classify grayscale images of clothing items into one of ten categories.

# Why CNNs are Better than ANNs for Image Data

Artificial Neural Networks (ANNs) treat every pixel as an independent input, resulting in a large number of parameters and ignoring the spatial relationships between pixels.

Convolutional Neural Networks (CNNs) are specifically designed for image processing. CNNs automatically extract important visual features such as edges, textures, and shapes using convolution operations. They require fewer parameters, train faster, reduce overfitting, and achieve much higher accuracy on image classification tasks.

---

# Purpose of Convolution and Pooling Layers

### Convolution Layer

The convolution layer applies filters (kernels) to the input image to detect important features such as:

- Edges
- Corners
- Textures
- Shapes

These extracted features help the model recognize objects in images.

### Max Pooling Layer

The pooling layer reduces the size of feature maps while preserving the most important information.

Benefits include:

- Faster computation
- Reduced memory usage
- Lower risk of overfitting
- Better generalization

---

# Model Architecture

The CNN model consists of the following layers:

```
Input Image (28×28×1)
        │
        ▼
Conv2D (32 Filters, 3×3, ReLU)
        │
        ▼
MaxPooling2D (2×2)
        │
        ▼
Flatten
        │
        ▼
Dense (128 Neurons, ReLU)
        │
        ▼
Output Layer (10 Neurons, Softmax)
```
