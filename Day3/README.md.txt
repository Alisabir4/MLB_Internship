# Day 3 – Python Data Structures & Student Record Management System

## Overview

On Day 3, I learned Python's core data structures and practiced solving programming problems using Lists, Tuples, Sets, and Dictionaries. I also built a Student Record Management System to apply these concepts in a real-world scenario.

---

## Learning Objectives

- Understand Python data structures.
- Learn when and where to use Lists, Tuples, Sets, and Dictionaries.
- Practice problem-solving using Python.
- Build a console-based application using Python fundamentals.

### Lists

### Tuples
- Count occurrences of an element
- Convert a tuple into a list and vice versa

### Sets
- Find unique values from a list
- Perform union and intersection operations

### Dictionaries
- Create a student record dictionary
- Calculate average marks of students
