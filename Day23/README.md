# Computer Vision Image Processing Studio

## Overview

Computer Vision Image Processing Studio is a web application built with **Python**, **OpenCV**, and **Streamlit**. It allows users to upload an image, apply different image processing techniques, preview the results, and download the processed image.

---

## Features

- Upload an image
- Display original image
- Display processed image
- Download processed image

---

## Image Processing Operations

- Grayscale
- Blur
- Edge Detection
- Image Rotation
- Image Enhancement
- Contour Detection
- Shape Detection

### Custom Features

- Image Flip
- Image Sharpening
- Thresholding

---

## Technologies Used

- Python
- Streamlit
- OpenCV
- NumPy
- Pillow

---

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/MLB_Internship.git
```

Go to the project folder:

```bash
cd Day23
```

Install the dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
