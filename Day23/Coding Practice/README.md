# Computer Vision Toolkit

A simple Computer Vision application built with **Python**, **OpenCV**, and **Gradio**. This app allows users to upload an image, apply different image processing operations, preview the result, and download the processed image.

## Features

* Upload an image
* Select an operation from a dropdown menu
* View the processed image
* Download the processed result

## Supported Operations

* Grayscale
* Blur
* Edge Detection
* Image Rotation
* Image Enhancement
* Contour Detection
* Shape Detection

## Technologies Used

* Python
* OpenCV
* Gradio
* NumPy

## How to Use

1. Upload an image.
2. Choose an image processing operation from the dropdown.
3. Wait for the processed image to appear.
4. Download the processed image if needed.
