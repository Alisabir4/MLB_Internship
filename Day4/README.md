# Day 4 - Conditional Statements, Loops & Logic Building

## Objective

The objective of Day 4 was to strengthen my understanding of Python fundamentals by practicing conditional statements, loops, and logic-building problems. I also developed a Number Analysis Tool and a Menu-Driven Application to apply these concepts in real-world scenarios.

---

## Concepts Learned

### Conditional Statements

* if statement
* if-else statement
* if-elif-else statement
* Nested conditions

### Loops

* for loops
* while loops
* range() function
* Iteration and repetition

### Functions

* Creating functions
* Calling functions
* Returning values
* Reusable code

### Logic Building

* Prime number checking
* Fibonacci sequence generation
* Palindrome checking
* Number reversal
* Digit counting

---

## Problems Solved

### Conditional Statements

1. Check whether a number is positive, negative, or zero.
2. Check whether a number is even or odd.
3. Create a grade calculator based on marks.
4. Find the largest among three numbers.
5. Check whether a year is a leap year.

### Loops

1. Print numbers from 1 to 100.
2. Print all even numbers from 1 to 100.
3. Calculate the sum of numbers from 1 to N.
4. Print the multiplication table of a given number.
5. Count the number of digits in a number.

### Logic Building Problems

1. Reverse a number.
2. Check whether a number is a palindrome.
3. Generate the Fibonacci sequence.
4. Check whether a number is prime.
5. Find all prime numbers between 1 and 100.

## Mini Project: Number Analysis Tool

### Features

* Accepts a number from the user.
* Checks whether the number is Even or Odd.
* Checks whether the number is Prime.
* Counts the number of digits.
* Reverses the number.
* Checks whether the number is a Palindrome.
* Displays all results in a user-friendly format.
